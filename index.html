Pilih store, lalu tentukan apakah hanya item OOS (SOH Qty 0) atau semua item dalam store tersebut.

Semua item pada store ini akan ditarik ke file export.
SOH Qty 0 = hanya item OOS, All = semua item pada store terpilih.

Export Item

Disediakan 2 versi export: Export Report untuk format ringkasan seperti sekarang tanpa warna, dan Raw Data untuk baris data database sesuai filter user.

Export Report = format ringkasan seperti saat ini tanpa warna. Raw Data Database = baris data asli sesuai filter.
Logic filter tidak diubah. Yang berubah hanya pilihan versi export.
